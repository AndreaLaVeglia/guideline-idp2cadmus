# Guida galattica al'utilizzo di Cadmus per NDP

In questa repository si può consultare tutto il materiale utile per lavorare nell'ambiente di editing di Cadmus Naples Dante Project (https://cadmus.naplesdante.it/items).

1. Per immettere nuovi items si utilizzi: 
[Guida pratica all'utilizzo di Cadmus Markdown-HTML](./tutorialForCadmus.md)  
[Guida pratica all'utilizzo di Cadmus PDF]

2. Se si tratta di un manoscritto già presente in Illuminated Dante Project (https://www.dante.unina.it/idp/public/pagine/manoscritti)che deve essere trasferito nel nuovo database, si utilizzino le Linee guida per il trasferimento di schede da IDP [(versione Markdown-HTML)](./guidelines-idp2cadmus-ITA.md) [(versione formato PDF)](./guidelines-idp2cadmus-ITA.pdf).
